# Hourly

## Install

```bash
$ npm i
$ npm i -g grunt-cli
$ grunt download-atom-shell
```


## Run

```bash
$ grunt && ./atom-shell/Atom.app/Contents/MacOS/Atom .
```

## Used Libraries

* atom-shell
* 6to5
* grunt
* browserify
* pouchdb
