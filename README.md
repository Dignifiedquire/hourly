# Hourly

## Install

```bash
$ npm i
$ npm i -g gulp
$ grunt download-atom-shell
```
and install [flow](flowtype.org).


## Run

```bash
$ gulp
$ ./vendors/atom-shell/Atom.app/Contents/MacOS/Atom .
```

## Used Libraries

* atom-shell
* 6to5
* grunt
* browserify
* pouchdb
* flow
* bootstrap3
* font-awesome
* less
