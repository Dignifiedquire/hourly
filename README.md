# Hourly

## Install

```bash
$ npm i
$ npm i -g gulp
$ grunt download-atom-shell
```
and install [flow](flowtype.org).


## Run

```bash
$ gulp
$ ./vendors/atom-shell/Atom.app/Contents/MacOS/Atom .
```
